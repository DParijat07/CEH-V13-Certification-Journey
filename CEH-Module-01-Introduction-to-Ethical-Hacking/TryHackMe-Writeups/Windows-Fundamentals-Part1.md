# Key Concepts Learned

## Windows Operating System

Windows is a commercial and closed-source operating system developed by Microsoft.

It is widely used in:

* enterprises
* personal systems
* corporate environments
* Active Directory infrastructures

---

# Windows File System

## NTFS (New Technology File System)

Features:

* file permissions
* encryption support
* access control
* journaling

---

# Important Windows Directories

## System32

Location:

```text
C:\Windows\System32
```

Contains critical system files and utilities.

---

# User Accounts

Windows supports:

* Administrator accounts
* Standard users
* Guest accounts

User privileges determine access levels within the system.

---

# Task Manager

Used for:

* monitoring running processes
* checking CPU and memory usage
* ending tasks
* startup management

Shortcut:

```text
Ctrl + Shift + Esc
```

---

# Control Panel

Used to manage:

* programs
* users
* network settings
* system configurations

---

# Windows Security Features

## Windows Defender

Built-in antivirus and security solution.

## Firewall

Filters incoming and outgoing network traffic.

## User Account Control (UAC)

Helps prevent unauthorized changes.
