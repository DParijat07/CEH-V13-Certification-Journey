# Nmap Practice

### Command Practiced

```bash
nmap -sV <IP Address>
```

### Purpose

* Detect running services
* Identify service versions
* Gather information about target systems
* Assist in vulnerability assessment

### Example

```bash
nmap -sV 192.168.x.x
```

### Key Observation

Service version detection is important because outdated services may contain known vulnerabilities that can be identified during a security assessment.
