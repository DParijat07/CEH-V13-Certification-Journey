# Lab Setup

This directory contains virtualization, Linux, VPN, and networking lab setup documentation for my cybersecurity learning journey.

# Cyber Security Homelab Setup

This directory contains the complete documentation of my cybersecurity homelab used for:

- CEH v13 practice
- Red Teaming
- Blue Teaming
- SOC monitoring
- TryHackMe labs
- Hack The Box labs
- Vulnerability analysis
- SIEM practice

---

# Lab Goals

- Learn practical ethical hacking
- Understand attack and defense techniques
- Practice enumeration and exploitation
- Analyze logs using SIEM
- Build hands-on cybersecurity skills

---

# Core Technologies

- VMware Workstation
- Kali Linux
- Windows 7
- Metasploitable 2
- Ubuntu Server
- Wazuh SIEM

---

# Lab Architecture

- Kali Linux → Attacker Machine
- Windows 7 → Windows Victim
- Metasploitable 2 → Vulnerable Linux Target
- Ubuntu + Wazuh → SOC/SIEM Monitoring Server

---

# Network Design

- VMware NAT Adapter → Internet + VPN
- VMware Host-Only Adapter → Internal Lab Network

---

# Important Safety Rules

- Vulnerable machines are isolated from the real network
- Malware testing is restricted to lab environments only
- Snapshots are used before major attacks
- VPN access is only used from Kali Linux
