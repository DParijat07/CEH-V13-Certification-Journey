# Kali Linux

## Overview

Kali Linux serves as the primary attacker machine in my cybersecurity home lab.

## Purpose

* Network Scanning
* Enumeration
* Vulnerability Assessment
* Penetration Testing Practice
* CEH Lab Exercises
* TryHackMe Learning

## Common Tools

* Nmap
* Netdiscover
* ARP-Scan
* Metasploit Framework
* Wireshark
* Burp Suite
* Hydra

## Role in Lab

Kali Linux is used to identify, analyze, and assess systems within the virtual lab environment while following ethical and authorized testing practices.
