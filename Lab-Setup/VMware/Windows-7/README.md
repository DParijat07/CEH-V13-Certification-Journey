# Windows 7

## Overview

Windows 7 is maintained as a target machine within the cybersecurity lab for learning and educational purposes.

## Purpose

* Network Scanning Practice
* Service Enumeration
* Vulnerability Analysis
* CEH Practical Labs
* Security Testing Demonstrations

## Learning Areas

* Host Discovery
* Port Scanning
* Service Detection
* Vulnerability Identification
* Security Assessment Methodology

## Role in Lab

Windows 7 provides a controlled environment for understanding Windows-based systems and common security weaknesses in a safe laboratory setting.
