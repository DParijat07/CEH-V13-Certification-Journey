# CEH Module 01 - Introduction to Ethical Hacking

This module contains foundational concepts of cybersecurity and ethical hacking based on the Certified Ethical Hacker (CEH) v13 syllabus.

The purpose of this module is to build a strong understanding of:

* cybersecurity fundamentals
* ethical hacking concepts
* hacker methodologies
* Linux basics
* virtualization
* cybersecurity lab environments

---
