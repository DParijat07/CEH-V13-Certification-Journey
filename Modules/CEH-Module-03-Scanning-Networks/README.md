# CEH Module 03 - Scanning Networks

## Overview
This module focuses on the network scanning phase of ethical hacking. The goal is to identify live hosts, open ports, running services, operating systems, and potential vulnerabilities within a target environment.

## Learning Objectives
- Understand network scanning concepts
- Learn host discovery techniques
- Perform port scanning
- Identify services and versions
- Understand OS fingerprinting
- Explore vulnerability discovery basics
- Document findings professionally

## Topics Covered
- Host Discovery
- Port Scanning
- Service Enumeration
- Banner Grabbing
- OS Detection
- Nmap Scanning Techniques
- Vulnerability Identification

## Practical Activities
- Nmap Scanning
- Enumeration Exercises
- TryHackMe Labs
- Home Lab Practice

## Tools Used

Understand how attackers and penetration testers discover and enumerate systems before exploitation.
