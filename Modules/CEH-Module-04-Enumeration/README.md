# CEH Module 04 - Enumeration

## Overview

Enumeration is the process of actively extracting information from discovered services, systems, and network resources. It is a critical phase of ethical hacking that follows scanning and helps identify usernames, shares, services, domains, and potential attack paths.

## Learning Objectives

- Understand enumeration concepts
- Identify service vulnerabilities
- Enumerate SMB, FTP, SSH, SMTP, DNS, RDP, and VNC services
- Discover users, shares, and resources
- Learn credential attack concepts
- Understand vulnerability identification workflows
- Gain hands-on experience with Hydra and Metasploit
