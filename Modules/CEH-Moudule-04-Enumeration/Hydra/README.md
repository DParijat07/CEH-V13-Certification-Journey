# Hydra

## Objective

Learn how credential attacks work against network services in authorized lab environments.

## Topics Covered

- Password Attacks
- Dictionary Attacks
- Username Discovery
- Service Authentication Testing

## Supported Services Studied

- FTP
- SSH
- SMB
- Telnet
- RDP

## Documentation Format

### Target Service

### Attack Method

### Configuration

### Results

### Observations

### Lessons Learned

## Important Note

Hydra should only be used in authorized environments such as personal labs, training platforms, and penetration tests performed with permission.
