# SMB Enumeration

## Objective

Understand how SMB services expose information about systems, users, domains, and shared resources.

## Topics Covered

- SMB Fundamentals
- NetBIOS
- SMB Shares
- User Enumeration
- Domain Information
- Null Sessions

## Tools

- Nmap
- SMBClient
- Enum4Linux

## Documentation Format

### Target
Target information.

### Enumeration Commands
Commands executed.

### Findings
Users, shares, and resources discovered.

### Security Risks
Potential weaknesses identified.

### Lessons Learned
Key takeaways from the exercise.
