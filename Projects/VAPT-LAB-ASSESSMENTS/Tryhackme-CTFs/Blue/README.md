# Blue - VAPT Lab Assessment

## Overview

Blue is a Windows exploitation lab from TryHackMe focused on identifying, validating, and exploiting the MS17-010 (EternalBlue) vulnerability. This assessment demonstrates the complete penetration testing lifecycle from reconnaissance through post-exploitation.

---

## Skills Demonstrated

* Network Scanning
* Service Enumeration
* SMB Enumeration
* Vulnerability Assessment
* CVE Research
* Metasploit Framework
* Exploitation
* Post-Exploitation
* Technical Documentation
