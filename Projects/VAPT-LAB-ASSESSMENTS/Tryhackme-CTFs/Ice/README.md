I# ICE - VAPT Lab Assessment

## Overview

ICE is a Windows-based vulnerability assessment and exploitation lab from TryHackMe. The assessment focuses on discovering a vulnerable Icecast Media Server, performing vulnerability research, gaining initial access, escalating privileges, and exploring post-exploitation techniques.
